    دليل الرفع السريع - YouTube Downloader
═══════════════════════════════════════════════════════════════

📦 محتويات الفولدر:
✓ web_app.py         - الملف الرئيسي
✓ requirements.txt   - المكتبات المطلوبة
✓ Procfile          - ملف التشغيل
✓ runtime.txt       - إصدار Python
✓ templates/        - صفحات HTML
✓ static/           - الملفات الثابتة
✓ app.json          - إعدادات Heroku
✓ .gitignore        - ملفات Git المستثناة

═══════════════════════════════════════════════════════════════
🚀 الطريقة 1: Render.com (موصى بها - مجانية 100%)
═══════════════════════════════════════════════════════════════

1. سجل على: https://render.com
2. اضغط "New +" → "Web Service"
3. اربط GitHub أو ارفع الملفات مباشرة
4. الإعدادات:
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn web_app:app
5. اضغط "Create Web Service"

✅ مزايا:
   • مجاني بالكامل
   • FFmpeg مثبت مسبقاً
   • سهل الاستخدام

═══════════════════════════════════════════════════════════════
🚀 الطريقة 2: Heroku (تحتاج بطاقة ائتمان)
═══════════════════════════════════════════════════════════════

1. ثبت Heroku CLI من: https://devcenter.heroku.com/articles/heroku-cli
2. افتح Terminal في مجلد المشروع:

   heroku login
   heroku create youtube-downloader-app

   git init
   git add .
   git commit -m "Initial deployment"

   heroku buildpacks:add https://github.com/jonathanong/heroku-buildpack-ffmpeg-latest.git
   heroku buildpacks:add heroku/python

   git push heroku main
   heroku open

⚠️ ملاحظة: Heroku يحتاج بطاقة ائتمان حتى للخطة المجانية

═══════════════════════════════════════════════════════════════
🚀 الطريقة 3: PythonAnywhere (مجانية محدودة)
═══════════════════════════════════════════════════════════════

1. سجل على: https://www.pythonanywhere.com
2. اذهب إلى Files وارفع كل المجلد
3. افتح Bash Console:

   pip install --user -r requirements.txt

4. اذهب إلى Web → Add a new web app → Flask
5. اختر web_app.py كملف رئيسي

⚠️ ملاحظة: FFmpeg قد لا يكون متاحاً في الخطة المجانية

═══════════════════════════════════════════════════════════════
🚀 الطريقة 4: Railway.app (سهلة جداً)
═══════════════════════════════════════════════════════════════

1. سجل على: https://railway.app
2. اضغط "New Project" → "Deploy from GitHub"
3. اختر المشروع
4. Railway سيكتشف التطبيق تلقائياً ويرفعه

✅ مزايا:
   • سهل جداً
   • مجاني لحد معين من الاستخدام

═══════════════════════════════════════════════════════════════
🖥️ تشغيل محلي للتجربة
═══════════════════════════════════════════════════════════════

pip install -r requirements.txt
python web_app.py

افتح: http://localhost:5000

🆘 حل المشاكل الشائعة
═══════════════════════════════════════════════════════════════

❌ Application Error:
   → تحقق من الـ logs
   → تأكد من requirements.txt

❌ Build Failed:
   → تأكد من runtime.txt
   → تحقق من إصدار Python

❌ FFmpeg not found:
   → أضف buildpack لـ FFmpeg
   → أو استخدم Render (FFmpeg مثبت)
