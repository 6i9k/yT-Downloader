═══════════════════════════════════════════════════════════════
🚀 أسرع طريقة للبدء (2 دقيقة) - Railway.app
═══════════════════════════════════════════════════════════════

1. سجل على Railway.app
   👉 https://railway.app

2. اضغط "Start a New Project"

3. اختر "Deploy from GitHub repo"
   (أو اضغط "Empty Project" ثم ارفع الملفات)

4. اختر المجلد أو ارفع الملفات

5. Railway سيكتشف التطبيق تلقائياً!

6. ✅ تم! موقعك جاهز في 2-3 دقائق

💡 لا يحتاج إعدادات معقدة - كل شيء تلقائي!

═══════════════════════════════════════════════════════════════
🌐 طريقة بديلة: Vercel (للمواقع السريعة)
═══════════════════════════════════════════════════════════════

1. سجل على Vercel
   👉 https://vercel.com

2. اضغط "Add New" → "Project"

3. اختر "Import Git Repository" أو ارفع الملفات

4. Vercel سيرفع تلقائياً

5. ✅ تم!

═══════════════════════════════════════════════════════════════
🎨 طريقة للمبتدئين: Glitch.com (بدون Git)
═══════════════════════════════════════════════════════════════

1. اذهب إلى: https://glitch.com

2. اضغط "New Project" → "Import from GitHub"
   أو اضغط "glitch-hello-python"

3. ارفع الملفات مباشرة من المتصفح

4. اضغط "Show" لرؤية الموقع

5. ✅ تم!

═══════════════════════════════════════════════════════════════
🧪 تجربة محلية أولاً؟
═══════════════════════════════════════════════════════════════

على Windows:
1. اضغط مرتين على: start_server.bat
2. افتح: http://localhost:5000
3. جرب تحميل فيديو

على Mac/Linux:
1. افتح Terminal
2. pip install -r requirements.txt
3. python web_app.py
4. افتح: http://localhost:5000
