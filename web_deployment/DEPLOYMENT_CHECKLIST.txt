═══════════════════════════════════════════════════════════════
    قائمة التحقق قبل الرفع - Deployment Checklist
═══════════════════════════════════════════════════════════════

📋 قبل الرفع، تأكد من:

□ جميع الملفات موجودة:
  □ web_app.py
  □ requirements.txt
  □ Procfile
  □ runtime.txt
  □ templates/index.html
  □ static/icon.ico
  □ app.json
  □ .gitignore

□ الإعدادات صحيحة:
  □ PORT يقرأ من environment variable
  □ SECRET_KEY مضبوط
  □ المسارات نسبية (ليست مطلقة)
  □ لا توجد كلمات مرور أو مفاتيح API في الكود

□ الاختبار المحلي:
  □ تم تشغيل السيرفر محلياً بنجاح
  □ التحميل من YouTube يعمل
  □ التحميل من Instagram يعمل (إذا أمكن)
  □ التحميل من Facebook يعمل (إذا أمكن)
  □ الواجهة تظهر بشكل صحيح
  □ Progress bar يعمل

□ المتطلبات:
  □ Python 3.11+ مثبت
  □ جميع المكتبات في requirements.txt
  □ FFmpeg متوفر (على السيرفر)
  □ Git مثبت (للرفع)

═══════════════════════════════════════════════════════════════
✅ خطوات الرفع على Render.com
═══════════════════════════════════════════════════════════════

1. □ إنشاء حساب على https://render.com
2. □ إنشاء repository على GitHub (اختياري)
3. □ رفع الملفات على GitHub أو Render مباشرة
4. □ إنشاء Web Service جديد
5. □ إعداد Build Command: pip install -r requirements.txt
6. □ إعداد Start Command: gunicorn web_app:app
7. □ اختيار Free Plan
8. □ الضغط على Create Web Service
9. □ انتظار اكتمال الـ deployment
10. □ اختبار الموقع المرفوع

═══════════════════════════════════════════════════════════════
✅ خطوات الرفع على Heroku
═══════════════════════════════════════════════════════════════

1. □ تثبيت Heroku CLI
2. □ تسجيل الدخول: heroku login
3. □ إنشاء تطبيق: heroku create APP_NAME
4. □ إضافة FFmpeg buildpack
5. □ إضافة Python buildpack
6. □ Git init & commit
7. □ رفع: git push heroku main
8. □ فتح التطبيق: heroku open
9. □ مراقبة logs: heroku logs --tail
10. □ اختبار جميع الوظائف

═══════════════════════════════════════════════════════════════
🔧 إعدادات Environment Variables
═══════════════════════════════════════════════════════════════

المتغيرات المطلوبة (على السيرفر):

□ PORT (تلقائي في معظم المنصات)
□ SECRET_KEY (للأمان)

اختياري:
□ DOWNLOAD_PATH (مسار التحميل)
□ MAX_FILE_SIZE (حد أقصى لحجم الملف)

═══════════════════════════════════════════════════════════════
🧪 اختبار ما بعد الرفع
═══════════════════════════════════════════════════════════════

□ الصفحة الرئيسية تفتح بنجاح
□ تحميل فيديو YouTube يعمل
□ تحميل فيديو Instagram يعمل
□ تحميل فيديو Facebook يعمل
□ تحميل Playlist يعمل
□ تحميل Audio فقط يعمل
□ اختيار الجودة يعمل
□ اختيار الصيغة يعمل
□ Progress bar يظهر بشكل صحيح
□ رسائل الخطأ تظهر بوضوح
□ الموقع responsive على الموبايل
□ السرعة مقبولة

═══════════════════════════════════════════════════════════════
⚠️ مشاكل شائعة وحلولها
═══════════════════════════════════════════════════════════════

❌ Application Error / 500 Error:
   ✓ تحقق من الـ logs
   ✓ تأكد من تثبيت جميع المكتبات
   ✓ تحقق من الـ PORT

❌ H10 - App Crashed (Heroku):
   ✓ تحقق من Procfile
   ✓ تأكد من gunicorn مثبت
   ✓ راجع heroku logs --tail

❌ Build Failed:
   ✓ تحقق من runtime.txt
   ✓ تأكد من Python version مدعوم
   ✓ راجع requirements.txt

❌ FFmpeg not found:
   ✓ أضف FFmpeg buildpack
   ✓ أو استخدم Render (مثبت مسبقاً)

❌ Download fails:
   ✓ تحقق من yt-dlp version
   ✓ تأكد من الرابط صحيح
   ✓ تحقق من السيرفر يدعم التحميل

❌ Slow performance:
   ✓ استخدم خطة أفضل
   ✓ قلل جودة التحميل
   ✓ تحقق من سرعة الإنترنت
